# Laptop Price Prediction

## Overview
This project predicts laptop prices using Machine Learning regression models.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## Models Used
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor
- CatBoost Regressor

## Features
- Data Cleaning
- Feature Engineering
- Encoding & Scaling
- Model Evaluation
- Interactive Prediction App

## Results
The best model achieved a high R² Score for laptop price prediction.
